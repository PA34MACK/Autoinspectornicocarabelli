# Dashboard

Dashboard personal para seguimiento de trabajos de CVAT.

## Funciones

- Registro de trabajos
- Cronómetro Pomodoro
- Estadísticas
- Seguimiento de estados
- Exportar / Importar Backup JSON
- GitHub Pages

## Backups

Los respaldos se almacenan en la carpeta

backups/

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages
